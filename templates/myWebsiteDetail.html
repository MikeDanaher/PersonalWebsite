Dynamic Login Home Page

Details

When desiging this page, I had to take into consideration the fact that several different groups would be accessing this page. Each group was to see different content; however, some users could fall into more than one group. Therefore, to create this page, I needed to make every item dynamic. If you were supposed to see something, it would appear. Otherwise, it wouldn't even get rendered when the page loaded.

To ensure that items appeard correctly, I had to create a hierarcy of users. Content for users higher in the stack was rendered first. Unfortunately, becuase of the limitations of the CMS there is a good amount of duplication. If content is only slightly different (say a link to other content) I had to create a completely new content box. This makes it difficult when content changes.

Overall, the performace is pretty good and so far all users are seeing just the content they should. It's been a great tool to give specific content to specific users without having to create many different duplicate pages
